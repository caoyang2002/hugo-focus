# 10. 评论

打开 Giscus 进行配置： https://giscus.app/

[中文页面](https://giscus.app/zh-CN)

## 配置

```bash
[params.giscus]
repo = 'user_name/your_repo'
repoId = 'your_repo_id'
category = 'your_category_name'
categoryId = 'your_category_id'
mapping = 'path_name'
strict = "0"
reactionsEnabled = "1"
emitMetadata = "0"
inputPosition = "top"
crossorigin = "anonymous"
theme = 'light'
lang = 'zh-CN'
```
